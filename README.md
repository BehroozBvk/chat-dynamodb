**Installation**

_Step 1:_

```bash
python3 -m venv .venv
```

_Step 2:_

```bash
source ./.venv/bin/active
```

_Step 3:_

```bash
pip3 install -r requrements.txt -t ./.venv/lib/python3.11/site-packages
```

_Step 4:_

```bash
python3 server.py
```
